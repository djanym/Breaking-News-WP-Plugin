== Description ==
Adds ability to mark a post with "Breaking news" tag to display it on all website pages.

== Installation ==
1. Copy plugin folder to wp-content/plugins folder
2. Activate plugin via Plugins page in admin
3. Open edit post page in admin and check breaking news options section
4. If your theme has <header> tag then the active breaking news post section will be added automatically.
Also, you can manually insert in a theme template this shortcode: <?php echo do_shortcode('[breaking_news]'); ?>
